# Article Publishing Workflow - Quality Checklist

**Purpose:** Prevent common errors in article publishing workflow.

## Pre-Publishing Checklist (MANDATORY)

### Amazon Links
- [ ] Amazon link is a **clickable hyperlink** (`<a href="...">text</a>`), NOT plain text
- [ ] Link uses `target="_blank" rel="noopener noreferrer"` for external links
- [ ] Link appears in first 2 paragraphs
- [ ] ASIN is correct and matches the book being reviewed

### Internal Links
- [ ] ONLY link to **directly related articles** (same category/topic)
- [ ] NO links to unrelated articles just to fill space
- [ ] Related Articles section should have 1-3 truly relevant links max
- [ ] Internal links use relative paths correctly

### Content Quality
- [ ] Article body matches the topic (no reused content from other niches)
- [ ] Word count 800-1200 words
- [ ] Straight apostrophes only (')
- [ ] No banned AI vocabulary

### Technical
- [ ] JSON-LD URLs match canonical domain
- [ ] Meta description present
- [ ] Title matches topic

## What NOT to Do

### DON'T: Add unrelated internal links
❌ Linking to "Antique Tools" or "Fountain Pens" in a sewing machine article  
✅ Only link to directly related articles (e.g., Singer Featherweight in a Singer 301 article)

### DON'T: Leave Amazon links as plain text
❌ `You can find it on Amazon here: https://www.amazon.com/dp/...`  
✅ `You can find it on <a href="https://www.amazon.com/dp/..." target="_blank" rel="noopener noreferrer">Amazon</a>`

### DON'T: Force unrelated cross-links
❌ Adding links just to hit a quota  
✅ Only include links that genuinely help the reader

## Related Article Guidelines by Category

### Sewing Machines
- Link to: Other sewing machine articles ONLY
- Don't link to: Tools, bikes, pens, etc.

### Bicycles
- Link to: Other bike/transportation articles ONLY
- Don't link to: Sewing machines, pens, toys, etc.

### Electronics
- Link to: Other electronics/tech restoration articles
- Don't link to: Completely unrelated categories

### General Rule
If the reader wouldn't naturally want to read the linked article next, DON'T link it.
